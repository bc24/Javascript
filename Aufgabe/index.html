Aufgabe 1

Erstelle eine HTML JavaScript Dokument. Füge diesem Dokument 3 Eingabefelder hinzu.
Die Eingabefelder: Vorname, Nachname, Alter
Füge einen Button "hinzufügen" hinzu.
Beim Klicken auf den Button soll ein Personenobjekt erstellt und einem Array hinzugefügt werden.

Füge der Seite ein Textfeld (p).
Füge einen Button "Anzeigen" hinzu, beim Klicken des Buttons sollen alle Personennamen in dem Textfeld angezeigt werden
